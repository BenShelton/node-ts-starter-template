# node-ts-starter-template

- node.js starter template

Get up to speed with nodejs development using this template with useful libraries and config:

- typescript
- ts-jest
- eslint
- airbnb
- commitlint
- husky
- prettier
- coveralls
- vscode
- travis
- typedoc
- github pages

## Installation

npm install jaspenlind/node-ts-starter-template
